# 🎵 Octave: Premium Audio Experience
**The Ultimate Client-Side Music Interface**

![Octave Logo](https://img.icons8.com/color/96/000000/music--v1.png)

**Octave** is a sleek, glassmorphism-inspired, high-performance web music player designed for the modern user. It acts as a lightweight, client-side interface that bridges the gap between public audio APIs and a premium, distraction-free listening environment. 

Whether you are looking for a **Spotify alternative**, a **fast song player alternative**, or a **minimalist audio streaming client**, Octave delivers a desktop-class experience right in your browser.

---

### 🚀 Key Features
* **✨ Glassmorphism UI**: High-end aesthetic with a "liquid" ambient background that color-matches the current album art.
* **📱 Mobile-First Design**: Optimized for touch, with zero-scroll layouts and perfectly placed controls.
* **🔒 Privacy-First**: 100% Client-side. Your playlists, history, and search data are stored locally in your browser's `localStorage`—no backend, no tracking.
* **🔍 API Power**: Built with robust Invidious and LRCLIB integrations for lyrics, artist bios, and media playback.
* **⏱️ Smart Features**: Built-in sleep timer, smart shuffle based on play-stats, and "Vault" export/import for your data.
* **⚡ Lightweight**: Zero-latency state management for a snappy experience.

---

### 🛠️ Architecture: How it works
Octave is a **purely client-side application**. 
* **Zero Backend**: There is no server hosting your data.
* **Direct Bridging**: Octave queries public Invidious APIs to fetch video/audio data in real-time.
* **Persistence**: All user preferences are handled via browser storage, ensuring that the app remains fast and decentralized.

### 🌐 Deploy Your Own Octave
Octave is designed to be hosted on any static site hosting provider. You can deploy it in under 5 minutes:

#### 1. GitHub Pages
1. Fork this repository.
2. Go to **Settings > Pages**.
3. Select the branch (main/master) and save. Your app will be live at `https://yourusername.github.io/octave/`.

#### 2. Cloudflare Pages / Netlify
1. Create a new project on your chosen platform.
2. Connect your GitHub repository.
3. Set the Build settings to defaults (no build step required for static files).
4. Click **Deploy**.
